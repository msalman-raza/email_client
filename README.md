# How to Run

Make sure you have following packages installed as Global
1. <b>Node</b>
2. <b>Gulp</b>
3. <b>Karma</b>

After cloning Run the following comman
1. <b>npm install</b>
2. <b>npm start</b>


You can also run following commands for different 
1. <b>npm run build</b> [ it will compile the code to dist folder]
2. <b>npm run lite</b> [ it will run the app on 3000 port, if you want to change port, pleas change in bs-config file]
3. <b>npm run unit</b> [ it will run the unit tests, and after that will prepare the coverage report]
4. <b>npm run report</b> [it will show coverage report on 3010 port, to change port change in coverage-bs-config.json ]
